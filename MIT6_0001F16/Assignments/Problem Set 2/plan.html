Functions Required

  • User Input User provide us with a letter and we must validate its a letter, if not they need to provide again.
  • get_guessed_word Tell the player if the letter guessed is in the word, reveal the letters guessed so far.
  • is_word_guess Tell the player if the word has been guessed
  • get_available_letters Show to the player the remaining letters that not used (they can select one.)
  • valid guess Tell the user if they have guessed not valid character or if they have already guessed the letter.

Variables

  • letters guessed - letters guessed so far, important check if word has been guessed, also tell user what have done so far.
  • number of guesses
  • number of warnings
  • secret word

Game Structure - Initialisation/Iteration

Initialisation

  1. Select the word
  2. Initialise the variables
    • number of guesses
    • number of warnings
    • letters guessed
    • available letters

Iteration/Loops

  1. Before each guess tell the user
    • If its the first guess, welcome the user and tell them how long the word is.
    • Available letters
    • Number of guesses remaining
  2. Ask user to guess a letter
  3. Check if guess valid
  4. Invalid guess
    1. Update either the warnings or guesses.
    2. Provide a message to if a warning lost and also what mistake was made.
    3. Show the guessed word
    4. Print some dashes to seperate the guesses.
    5. Go back to Step 1 (Show available letters and number guesses remaining)
  5. Valid guess
    1. Update letters guessed
    2. Game termination, word guessed Determine if the user has guessed the word. If so congratulate the user, reveal the word and the score.
    3. Word not guessed
      • Update word guess, available letters and guesses remaining
      • Determine if the letter guessed was in the secret word.
      • Provide a message if good guess or not
      • Show the guessed word.
      • Print some dashes to seperate the guesses.
      • Go back to step 1.
